# ChurrasApp

Aplicativo simples de controle de pedidos de marmitas para uma churrascaria local.

## Tecnologias utilizadas
- React Native (JSX e componentes básicos)
- Node.js e ambiente Visual Studio Code
- AsyncStorage para persistência local
- SQLite (pode ser integrado na etapa avançada)
- React Navigation para navegação entre telas

## Funcionalidades
- Cadastro de pedidos locais (sem entrega)
- Listagem de pedidos salvos
- Armazenamento local dos dados
- Interface simples e responsiva

## Como executar
1. Instale o Node.js e o Expo CLI: `npm install -g expo-cli`
2. Instale as dependências: `npm install`
3. Execute o projeto: `npm start`
4. Escaneie o QR Code com o aplicativo Expo Go no seu dispositivo Android.
